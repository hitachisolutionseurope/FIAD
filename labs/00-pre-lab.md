# Pre-lab: Connecting to Fabric

## 1. Connecting to Microsoft Fabric

Navigate to [https://app.fabric.microsoft.com/](https://app.fabric.microsoft.com/)

Connect with the credentials provided to you as part of the FIAD session.

![Access login](/images/prelab/Image2.png)
---

## 2. Access the Fabric Home Screen

You are given the option to create a workspace with pre-designed templates 
for different task flows. However, for the purpose of this event we will 
not use these.

![Fabric Home Screen](/images/prelab/Image3.png)

---

## 3. Selecting the Correct Workspace

Ensure that you are working in the correct Fabric Workspace.

From the left navigation pane, click the Workspaces button and select the 
FIADStudent Workspace that corresponds to your student id.

> **Warning: Do not use the "My Workspace" environment.**

![Workspace selection](/images/prelab/Image4.png)
